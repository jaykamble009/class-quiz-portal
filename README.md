# 🎓 Class-Quiz Portal: Secure Assessment Terminal

A professional-grade, high-tech assessment platform for teachers and students. Built with **React**, **Vite**, and **Supabase**, this portal offers a seamless and secure environment for conducting digital examinations.

## ✨ Key Features
- **Elite Proctoring**: Real-time student monitoring with security strike detection.
- **Smart Dashboard**: Comprehensive control panel for teachers to manage exams, students, and subjects.
- **AI Integration**: Built-in support for generating assessment content.
- **Responsive Design**: Optimized for laptops and desktops with a "fit-to-screen" assessment interface.
- **Secure Architecture**: Robust authentication and Row Level Security (RLS) powered by Supabase.

## 🚀 Quick Start

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Configure Environment**:
   Create a `.env.local` file with your credentials:
   ```env
   VITE_SUPABASE_URL=your_project_url
   VITE_SUPABASE_ANON_KEY=your_anon_key
   VITE_GEMINI_API_KEY=your_gemini_key
   ```

3. **Run Dev Server**:
   ```bash
   npm run dev
   ```

## 🛠 Tech Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend / DB**: Supabase (PostgreSQL)
- **Bundler**: Vite
- **Icons**: FontAwesome 6 Pro
